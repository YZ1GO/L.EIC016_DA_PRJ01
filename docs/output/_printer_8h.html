Detailed Description

Contains functions for exporting and printing data related to a water distribution system.

This file defines functions for exporting reservoir, station, and city data to text files, as well as functions for printing various metrics related to the water distribution system.

The functions provided here facilitate the analysis and visualization of data associated with the water distribution system, aiding in understanding its performance and characteristics.
